class Team_1_box_and_button.Box(msg, x, y, w, h, c, s=0)[source]¶

Bases: object

This class is used to produce boxes to represent information to players

Parameters:
  • msg – Message to appear in the box.
  • x (int) – x coordinate of the top left-hand corner of the box.
  • y (int) – y coordinate of the top left-hand corner of the box.
  • w (int) – width of the box.
  • h (int) – height of the box.
  • c (str) – colour of the box
  • s (int) – size of the border line of the box, default 0.
