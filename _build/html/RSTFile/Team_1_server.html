Team_1_server module¶

A Server that runs a lobby for Ludo Games

The server listens for any connections to any clients. It allows players to have multiple games running instantaneously.

A class that keeps a list of current client connections. This allows the threads, which are created by each connection, to broadcast a message to all connections.
