Team_1_player module¶

class Team_1_player.Player(colour, name, all_pieces, names)[source]¶

Bases: object

Create an object to represent a player with 4 piece objects.

The function of this class is limited to representing attributes for the player; therefore there are no methods. This class also holds universal attributes, like all_pieces and names that are used to draw pieces and scores that don’t belong to this client and therefore wouldn’t have access to.

Parameters:
  • colour (str) – The colour used to represent player and all their pieces.
  • name (str) – The name displayed with player’s score and chat messages.
  • all_pieces (list) – A list of all Piece objects in the game currently.
  • names (list) – The list of all player names. Needed to display the score board.
