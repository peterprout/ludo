Team_1_client module¶

class Team_1_client.Ludo[source]¶

Bases: object

This is the main Ludo class.

It initialises my_player, genie_owner, all_pieces, board, connection, the score board and the timer.

It also holds the main run function which runs the game.

Variables:
  • self.my_player – Instance of Player object.
  • self.genie_owner – Value representing owner of genie.
  • self.starting_point – List represent starting position of each player
  • self.colour_list – List of colours.
  • self.colour_to_img – Image representing colour
  • self.all_pieces – List of all pieces
  • self.board – Instance of board object
  • self.connection – Instance of connection
  • self.current_player – Current player
  • self.colour_check – Check if flashing.
  • self.time_limited – Amount of time before automatic movement/roll
  • self.p – Value from server for time function to activate.
  • self.font – Type of font
  • self.text – Text of time function
bgm()[source]¶

Sets the frequency, loads the background music and plays it.

check_click(x_value, y_value)[source]¶

If there is a click after a dice has been rolled check all my_player pieces. If piece is movable check the size of the piece to set the clickable coordinates.

Parameters:
  • x_value (int.) – The x_value coordinate of the click.
  • y_value (int.) – The x_value coordinate of the click.
click_piece(num)[source]¶

After a dice is rolled, if the player clicks a movable piece, call click_piece. It calls the move_piece function, it also sends what piece was moved to the server.

Parameters:num (int.) – the number of the piece.
draw_time_out()[source]¶

Draws the timer which counts down until it reaches 0. When this happens it goes back to its original number and counts down again.

pause()[source]¶

Pauses all music and sounds.

run()[source]¶

This is the main game method. It draws the board, pieces and the buttons. It also shows the diceS rolling animation.

setup()[source]¶

Creates the coo-rdinate dictionary for the board, initialises pygame. It also blocks out some pygame events so the queue doesn’t overflow from unneeded events such as MOUSEMOTION. It also setups attributes in board, connects to the server and shows the start screen.

show_start_screen()[source]¶

Shows the start screen whent game first starts.

It shows the word LUDO in flashing colours. Once the player connects to the server, the screen goes away.

terminate()[source]¶

Quit game if user closes window.

unpause()[source]¶

Unpauses all music and sounds.
