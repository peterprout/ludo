Team_1_connection module¶

class Team_1_connection.Connection(board, my_player, current, all_pieces)[source]¶

Bases: object

Connection is used to receive all communication from the server to this client and vice versa. All dynamic action by client occurs through Connection.py.

Parameters:
  • board (instance object) – The board object
  • my_player (instance object) – The player object
  • current (str) – Current player
  • all_pieces (list) – Array of all pieces.
connect_to_server()[source]¶

Connects client to server, creates thread to listen for incoming message.

connection_handler()[source]¶

This function controls all data received from the server, and updates the client-side program according to the received JSON messages.

When referring to JSON message comments, if the symbols <> are used, it implies that the data is dynamic, and what will be in there depends on the player colour, roll of the dice etc.

end_roll()[source]¶

Called when player has finished movement of piece. Resets all player’s pieces.movable to False. Checks if player should end turn, Otherwise allows player another dice roll. Checks if all player’s pieces on home run, allowing player to win.

end_screen(names, scores, string)[source]¶

Creates a TKinter window which shows the scores of each player :param names: list of player names :param scores: list of scores :param string: text to be shown on scoreboard

end_turn()[source]¶

Called when player’s turn is over. It resets player turn token, the amount of rolls, and ability to roll dice. Sends message to server informing other players your turn is over.

Checks if any or all players pieces can be played.

time_out()[source]¶

This method is called when the timer has run out of time. It will automatically roll dice or move random piece when timer runs out.
